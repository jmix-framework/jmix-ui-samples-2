The Jmix UI Samples application contains a large collection of examples demonstrating Jmix UI in action. Each example includes source code, description and links to the documentation.

Components section demonstrates the usage of visual components designed to display and input data and interact with the user.

Layouts section shows examples of arranging components on a view.

Dialogs and notifications section contains examples of various standard dialogs and notifications.

Custom components section demonstrates different techniques of creating custom UI components.

Cookbook contains a collection of recipes for solving typical problems related to UI.
