DataGrid предназначен для отображения и сортировки табличных данных и предоставляет средства для управления строками.

В этом примере вы можете попробовать следующие настройки для DataGrid:

  • sortable - включает или отключает сортировку для column. Значени по умолчанию true. Если сортировка разрешена, то при нажатии на название колонки справа от названия появляется соответствующая иконка. Сортировку отдельно взятой колонки можно запретить с помощью атрибута sortable этой колонки.
  • resizable - включает или отключает изменение размера для column. Значение по умолчанию false.
  • columnReorderingAllowed - разрешает или запрещает пользователю менять местами колонки, перетаскивая их с помощью мыши. По умолчанию имеет значение true.
