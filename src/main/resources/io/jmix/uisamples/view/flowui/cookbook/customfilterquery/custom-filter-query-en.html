This example shows how to implement a custom filter by programmatically modifying the query text in the TypedValueChangeEvent handler of an input field.

See reload() method of CustomFilterQuerySample class. If the filter field is not empty, it adds a where condition to the query text and sets the appropriate query parameter value. The value contains the (?i) prefix which ensures case-insensitive search.

The DataLoadCoordinator facet is not used in this example. Instead, the reload() method is invoked on BeforeShowEvent explicitly in the controller.

Note also that genericFilter component can be used together with a custom filter.
