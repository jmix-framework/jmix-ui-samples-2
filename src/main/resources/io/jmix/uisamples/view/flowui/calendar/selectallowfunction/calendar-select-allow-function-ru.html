Функция разрешения выделения определяет выделение ячейки в целом. Данная функция принимает в качестве параметра объект, который содержит информацию о выделении:

  • start - дата начала с временной зоной компонента. startStr - дата начала в строковом формате. end - дата окончания с временной зоной компонента. endStr - дата окончания в строковом формате. allDay - определяет что ячейка является днём.
