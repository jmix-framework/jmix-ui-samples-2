You can create custom UI components in addition to the comprehensive suite of standard components and layouts described above.

The below examples demonstrate different techniques of creating custom components.

  • Composite component example shows how to create a reusable component for editing addresses. It contains of multiple standard UI components and encapsulates its own data loading logic.
  • Integrating a JavaScript library is an example of wrapping the jQuery UI Slider widget.
  • Creating the component from scratch explains how to create the light/dark theme toggle which you can see in the top right corner of this application.
