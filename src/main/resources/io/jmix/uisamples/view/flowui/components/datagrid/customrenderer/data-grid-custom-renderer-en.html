It is possible to customize the render of column.

The following renderers can be defined declaratively in XML:

  • localDateRenderer
  • localDateTimeRenderer
  • numberRenderer

E.g.:

<column property="dueDate"> <localDateRenderer format="yyyy-MM-dd"/> </column>

Alternatively, renderers can be supplied in a view controller using the @Supply annotation.

Method annotated with @Supply returns a value that will be passed as an input parameter to a method defined by subject.

Also, you can use the addColumn() method for it.
