Кнопка имеет несколько предопределённых тем:

  • primary
  • success
  • error
  • contrast
  • large
  • small
  • icon
  • tertiary
  • tertiary-inline

Вы можете применить эти стили в XML-дескрипторе используя атрибут themeNames или в контроллере экрана используя константы из класса ButtonVariant.
