The FullCalendar component generates a set of events. There are two events that are generated without specific conditions: DateClickEvent and DatesSetEvent.

DateClickEvent - the event is fired when a day cell or time cell is clicked.

DatesSetEvent - the event is fired after the calendar date range has been initially set or changed in some way, and the DOM of the component has been updated. The calendar dates can change any time the user does the following:

  • Navigates to next and previous.
  • Switches the display mode.
  • Clicks a navigation link (e.g. week or day).
