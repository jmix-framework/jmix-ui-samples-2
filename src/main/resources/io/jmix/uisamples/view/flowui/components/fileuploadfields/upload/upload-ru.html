Компонент upload позволяет пользователям загружать один или несколько файлов. Он отображает прогресс загрузки и статус каждого файла. Файлы можно загружать, нажимая на кнопку Загрузить файлы, или перетаскивая их в область компонента.

В данном примере можно загрузить до 3 файлов, каждый с максимальным размером 10 КБ. Имя загруженного файла будет отображаться под кнопкой компонента.

В этом примере используется MultiFileMemoryBuffer для обработки нескольких файлов и записи данных файла в набор буферов в памяти. Для загрузки и сохранения больших файлов рекомендуется использовать компонент FileStorageUploadField.
