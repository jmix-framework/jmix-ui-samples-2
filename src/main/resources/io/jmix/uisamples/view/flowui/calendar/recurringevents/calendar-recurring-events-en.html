The component will consider an event as recurring, if it has one of the following properties: recurringDaysOfWeek, recurringStartDate, recurringEndDate, recurringStartTime or recurringEndTime.

Please note that, by default, an event in the data provider is expanded into single events in the calendar. If the user changes a recurring event, the change will be applied to the corresponding single event in the data provider.

If a recurring event has a group ID property, all recurring events will be moved or resized together.
