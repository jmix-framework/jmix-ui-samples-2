Диаграмма "Найтингейла" - это круговая диаграмма, сочетающая в себе элементы лепестковой диаграммы и столбчатой диаграммы.

Свойства roseType определяет, будет ли отображаться диаграмма Найтингейла, которая различает данные по радиусу.

Существует два режима настроек:

  • RADIUS - используйте центральный угол, чтобы показать процент данных, радиус, чтобы показать размер данных.
  • AREA - все сектора будут иметь один и тот же центральный угол, размер данных отображается только через радиус.
