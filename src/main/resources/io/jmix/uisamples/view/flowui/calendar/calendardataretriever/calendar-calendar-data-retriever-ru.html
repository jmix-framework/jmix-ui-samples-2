CalendarDataRetriever является простым программным способом загрузки данных в календарь, например, из сервиса. Данный поставщик похож на делегат загрузки callbackDataProvider, но без установки соотношения свойств сущности и без его определения в XML. Метод onItemsFetch() вызывается в следующих случаях:

  • Когда пользователь переключается на режим отображения, который имеет больший диапазон дат, чем предыдущий.
  • Когда вызываются методы навигации у FullCalendar, например #navigateToNext().
  • Когда меняется временная зона у компонента.
