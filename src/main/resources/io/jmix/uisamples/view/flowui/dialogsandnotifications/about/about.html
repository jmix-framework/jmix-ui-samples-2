A dialog is a small popup window that displays some information and can request the user for selection or input. There are different types of dialogs:

  • Message dialog just shows a message to the user.
  • Option dialog shows a message and presents a set of options.
  • Input dialog shows a message and presents a set of input fields.
  • Background task dialog starts a background task and shows its progress.

Notifications are simple informational messages displayed in pop-ups.
