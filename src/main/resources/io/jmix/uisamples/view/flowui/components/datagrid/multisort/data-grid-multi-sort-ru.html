Для настройки множественной сортировки колонок в DataGrid используются три атрибута:

  • multiSort - включает или отключает множественную сортировку
  • multiSortOnShiftClickOnly - позволяет добавлять выбранную колонку в сортировку только при нажатой клавише Shift
  • multiSortPriority - определяет правило, по которому последующие колонки будут добавляться в сортировку. PREPEND - в начало, APPEND - в конец. По умолчанию - PREPEND.
