Каждый режим отображения имеет специфичные свойства, которые можно указать внутри элемента displayModeProperties.

Также компонент FullCalendar позволяет определить свойства, которые не указаны в рамках XML свойств. Данные свойства переопределяют существующую конфигурацию календаря для конкретного режима отображения. Например, в данном экране режим месячной сетки делает видимыми номера недель. Обратите внимание, что название свойства в элементе property должно соответствовать названию свойства из документации FullCalendar.js.
