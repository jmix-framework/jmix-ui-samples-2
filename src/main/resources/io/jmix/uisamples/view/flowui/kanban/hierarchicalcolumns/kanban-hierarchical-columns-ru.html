В компоненте доски Kanban поддерживается иерархичность столбцов. Для того, чтобы определить иерархичный столбец достаточно добавить столбец в уже существующий столбец.

Существует два режима отображения столбцов:

  • TABS - отображение столбцов по вкладкам
  • COLUMNS - отображение всех столбцов внутри родительского столбца

При использовании режима TABS Вы можете управлять выбранным столбцом по умолчанию с помощью атрибута selected.

Заметка: иерархичные столбцы не отображаются, если на доске Kanban определены зоны ответственности (swimlanes).
