Компонент chart поддерживает инкрементальное обновление данных, если он связан с источником данных.

Компонент отслеживает изменения, происходящие в контейнере данных (коллекции) и отправляет разницу данных на клиентскую часть.

В данном примере данные автоматически обновляются с помощью timer фасета.
