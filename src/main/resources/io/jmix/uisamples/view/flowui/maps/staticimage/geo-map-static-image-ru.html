Компонент GeoMap поддерживает отображение статических изображений. Для этого нужно использовать слой image. На выбор есть два источника:

  • imageStaticSource отображает статические ресурсы. Атрибут url позволяет указать URL до внешнего ресурса или до ресурса в приложении, см. Vaadin docs :: Loading Resources Также можно программно указать ресурс через StreamResource.
  • imageWmsSource отображает изображение поставляемое WMS сервисом.

В данном примере отображается изображение из ресурсов приложения. Для imageStaticSource и mapView нужно обязательно указать projection и extent.
