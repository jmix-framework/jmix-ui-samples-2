Wizard(мастер создания) - это компонент для пошагового заполнения данных.

В данном примере продемонстрирован wizard для создания сущности Employee (см. Employee.java).
Мастер создания открывается в диалоге по кнопке и состоит из трёх шагов:

  • Запонение основной информации о работнике
  • Заполнение адреса
  • Проверка введённой информации

Каждый шаг связан с данными, информация о которых хранится в основном экране WizardDialog. Некоторые шаги используют данные для записи, некоторые для чтения. Два первых шага имеют поля, которые будут провалидированы при перехода на следующий шаг.

Пользователь может в любой момент переключаться между шагами (если данные на текущем шаге корректны) с помощью соответствующих кнопок, которые меняют своё поведение и состояние в зависимости от текущего шага.

Основой для создания wizard послужил компонент tabSheet. Шаги релизованы как отдельные влкадки. Содержимое каждой вкладки реализовано как fragment.

Полезные ссылки:
  • Компонент Composite – реализация компонента AddressComponent, который используется на втором шаге
  • Фрагменты – составной компонент, который используется для создания шагов
