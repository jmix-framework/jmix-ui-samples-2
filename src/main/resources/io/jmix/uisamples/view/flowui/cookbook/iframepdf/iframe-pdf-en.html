This sample demonstrates possible approaches for displaying PDF files as a static resource using the IFrame component.
Static resources should be stored in the META-INF/resources folder. To make displaying such resources available, you need to configure the frame-ancestors policy directive. See IframeCustomSecurity.java.

This source code contains a directive that allows the use of resources if the requests is sent from demo.jmix.io.
More detailed information is available in the documentation for the directive: CSP: frame-ancestors - HTTP | MDN 6.
