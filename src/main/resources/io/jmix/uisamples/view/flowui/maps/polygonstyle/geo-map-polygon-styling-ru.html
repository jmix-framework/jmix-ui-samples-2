PolygonFeature позволяет стилизовать заливку и обводку полигона.

VectorLayer также может иметь стили. Стили слоя применяются для всех геометрических объектов в источнике. Если объект имеет свой собственный стиль, то стили из слоя применяться не будут.

PolygonStyle - опциональный вспомогательный класс. Он предоставляет методы стилизации, применимые только для Polygon.
