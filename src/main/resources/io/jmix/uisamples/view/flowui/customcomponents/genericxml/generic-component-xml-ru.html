Элемент XML component можно использовать для создания любого компонента пользовательского интерфейса с помощью FQN. Чтобы определить свойства компонента, можно использовать вложенные элементы property. Это может быть удобно, если вы подключаете аддон или интегрируете компонент JavaScript. Т.е. в тех случаях, когда полная интеграция компонента в фреймворк не нужна, а достаточно поддержать его использование в XML.
