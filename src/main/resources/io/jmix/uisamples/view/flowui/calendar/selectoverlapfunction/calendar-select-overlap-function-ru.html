Данный пример показывает как функция перекрытия событий определяет какие ячейки с событиями могут быть выделены. Функция вызывается для каждого события в ячейке, если она вернёт false, то выделение ячейки не произойдёт.

Также есть свойство selectOverlap. Если оно отключено, то ячейки, которые содержат события, выделяться не будут.
