GroupDataGrid является компонентом Grid, который поддерживает группировку данных по определённым свойствам сущности. Компонент включает в себя колонку группировки GroupDataGridColumn, которая отображает группировку данных и компонент для раскрытия строки группировки. По умолчанию, заголовок колонки содержит иконку группировки, которая управляет группировкой по нажатию.

Чтобы декларативно выполнить группировку данных, используйте элемент groupBy где нужно указывать, по каким колонкам будет группировка.
