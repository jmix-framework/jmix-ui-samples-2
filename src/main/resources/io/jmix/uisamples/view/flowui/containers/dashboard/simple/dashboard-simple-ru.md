Компонент `dashboard` предоставляет гибкую компоновку сетки для организации и отображения виджетов. Он позволяет
пользователям располагать контент в настраиваемом формате панели управления.

Поддерживаются следующие функции:

- **Адаптивная сеточная компоновка**: автоматическая настройка размещения виджетов в зависимости от доступного
  пространства
- **Управление столбцами**: настройка максимального количества столбцов с помощью атрибута `maximumColumnCount`
- **Размер виджетов**: управление размерами виджетов с помощью атрибутов `colspan` и `rowspan`
- **Разделы**: группировка связанных виджетов с помощью `dashboardSection`
- **Редактируемый режим**: пользователи могут перетаскивать виджеты для изменения компоновки панели мониторинга
- **Пользовательский контент**: каждый виджет может содержать любые компоненты пользовательского интерфейса Jmix

В этом примере демонстрируется панель мониторинга с тремя виджетами и разделом, содержащим несколько виджетов.
