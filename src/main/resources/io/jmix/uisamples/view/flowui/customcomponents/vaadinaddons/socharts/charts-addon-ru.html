Вы можете использовать любые сторонние аддоны для Vaadin из Vaadin Directory или из других источников. В этом примере показана интеграция аддона SO Charts.

Чтобы добавить аддон Vaadin в ваше приложение, добавьте репозиторий Vaadin Add-ons и зависимость на аддон в файле build.gradle проекта:

repositories {
    maven {
        url 'https://maven.vaadin.com/vaadin-addons'
    }
    // ...
}

dependencies {
    implementation 'org.vaadin.addons.so:so-charts:6.0.1'
    // ...
}
После этого вы сможете использовать компоненты аддона программно, см. например ChartsAddonSample.java. Чтобы использовать компоненты аддона декларативно в XML экранов, требуется выполнить несколько дополнительных шагов:
  1. Создать XSD для компонентов (см. chart.xsd).
  2. Реализовать загрузчики компонентов (см. ChartLoader.java).
  3. Зарегистрировать компоненты и их загрузчики в приложении (см. ChartComponentRegistration.java).
Примечание: В данном примере предоставляется только частичная поддержка компонентов аддона (не все компоненты и свойства описаны для использования в XML).
