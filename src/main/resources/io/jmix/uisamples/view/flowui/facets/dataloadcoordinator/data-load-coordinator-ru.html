Фасет dataLoadCoordinator предназначен для запуска загрузчиков данных и для декларативного связывания загрузчиков данных с контейнерами, визуальными компонентами и событиями жизненного цикла экрана.

Существует три режима работы:

  • Automatic режим активируется атрибутом auto. Загрузчики срабатывают BeforeShowEvent.
  • Manual режим использует вложенные элементы refresh для настройки момента запуска загрузчиков.
  • Semi-automatic режим является комбинацией automatic и manual режимов. Когда атрибут auto установлен в значение true, и определено несколько элементов refresh, для DataLoadCoordinator будут автоматически настроены все загрузчики, которые не имеют ручной настройки.
