This example demonstrates how to create DataGrid with a variable set of columns.

It is based on key-value entities and containers:

  • Data is provided as a list of KeyValueEntity instances.
  • KeyValueCollectionContainer is created dynamically and configured using addProperty() methods to match properties of key-value entities.
  • DataGrid is created dynamically with columns matching the container properties.

See DataGridVariableColumnsSample.java for details.
