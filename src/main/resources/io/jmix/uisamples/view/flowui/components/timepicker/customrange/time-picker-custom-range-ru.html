Вы можете указать доступное для выбора время с помощью атрибутов max и min. Если вы их установите, время, выходящее за пределы указанного промежутка, будет недоступно.

Интервал по умолчанию между элементами, отображаемыми в выпадающем списке - один час. Можно установить пользовательское значение шага.
