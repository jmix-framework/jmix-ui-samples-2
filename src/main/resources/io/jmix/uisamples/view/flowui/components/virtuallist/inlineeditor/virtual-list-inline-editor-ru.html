Пример того, как можно построить inline editor над сущностью при помощи VirtualList. Шаги, по которым стоит выстраивать inline editor списков:

  • В экране объявить VirtualList.
  • Для созданного VirtualList сгенерировать Renderer;
  • В Renderer генерировать карточки Vertical Layout, каждая карточка состоит из 3 элементов:
    • Avatar - опциональный элемент, стоит в начале Vertical Layout (карточки);
    • Info Layout - любый набор компонент; цель - донести информацию в карточке о деталях текущего элемента;
    • Button Layout - layout в котором находятся управляющие кнопки работы с текущим элементом;
  • Создать DetailView который редактирует выбранную сущность или создает новую;
  • На все управляющие кнопки добавить слушатель событий для управления CRUD;
  • Ваш inline editor в VirtualList готов.
