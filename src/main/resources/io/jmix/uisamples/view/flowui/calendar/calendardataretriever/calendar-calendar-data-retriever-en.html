The CalendarDataRetriever provides a simple way to programmatically set data for the calendar, such as from services. It is similar to the load delegate of the callbackDataProvider, but it does not require property mapping or XML definition.

  • When the user switches to a calendar display mode that has a wider date range than the previous one.
  • When FullCalendar's navigation methods are invoked (e.g. #navigateToNext()).
  • When the time zone of the component is changed.
