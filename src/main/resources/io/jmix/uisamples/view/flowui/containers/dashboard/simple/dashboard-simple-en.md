The `dashboard` component provides a flexible grid-based layout for organizing and displaying widgets. It allows users
to arrange content in a customizable dashboard format.

The following features are supported:

- **Responsive Grid Layout**: Automatically adjusts widget placement based on available space
- **Column Control**: Configure maximum number of columns with `maximumColumnCount` attribute
- **Widget Sizing**: Control widget dimensions using `colspan` and `rowspan` attributes
- **Sections**: Group related widgets together using `dashboardSection`
- **Editable Mode**: Users can drag and drop widgets to rearrange the dashboard layout
- **Custom Content**: Each widget can contain any Jmix UI components

This example demonstrates a `dashboard` with three widgets and a section containing multiple widgets.
