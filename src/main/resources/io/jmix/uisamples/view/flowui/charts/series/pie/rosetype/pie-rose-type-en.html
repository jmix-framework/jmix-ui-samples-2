The Nightingale's Rose chart is a circular graph that combines elements of a radar chart and a column graph.

roseType property configures whether to show as Nightingale chart, which distinguishes data through radius.

There are 2 optional modes:

  • RADIUS - use central angle to show the percentage of data, radius to show data size.
  • AREA - all the sectors will share the same central angle, the data size is shown only through radius.
