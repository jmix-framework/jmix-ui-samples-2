Scroller - контейнер, который позволяет прокручивать свое содержимое.

С помощью атрибута scrollBarsDirection можно задавать направление расположения вложенных компонентов:

  • VERTICAL
  • HORIZONTAL
  • BOTH
  • NONE

Атрибут scrollBarsDirection позволяет настраивать полосы прокрутки. Он может принимать значения horizontal, vertical - для прокрутки по горизонтали и вертикали соответственно, both - для прокрутки во всех направлениях. Установка значения none запрещает прокрутку в любом направлении.
