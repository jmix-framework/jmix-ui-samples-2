This section contains a collection of recipes for solving typical problems related to UI.

Cascading dropdowns example shows how to implement selection of two dependent entities.

Copying entities example demonstrates possible approaches to copying entities in list views.

Custom filter using declarative conditions example shows how to implement a custom filter by declaring a set of query conditions linked with an input field.

Custom filter using query modification example shows how to implement a custom filter by programmatically modifying the query text.

Using database view demonstrates usage of a database view in the UI list view.
