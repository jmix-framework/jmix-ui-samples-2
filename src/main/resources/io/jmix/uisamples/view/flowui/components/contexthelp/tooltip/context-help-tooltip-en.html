A tooltip is a customizable context tooltip.

Attributes:

  • focusDelay - the delay in milliseconds before the tooltip is opened on keyboard focus, when not in manual mode.
  • hideDelay - the delay in milliseconds before the tooltip is closed on losing hover, when not in manual mode. On blur, the tooltip is closed immediately.
  • hoverDelay - the delay in milliseconds before the tooltip is opened on hover, when not in manual mode.
  • manual - when true, the tooltip is controlled programmatically instead of reacting to focus and mouse events.
  • position - position of the tooltip with respect to its target.
  • text - the string used as a tooltip content.
