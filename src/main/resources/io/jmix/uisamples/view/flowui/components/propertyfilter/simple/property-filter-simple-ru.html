PropertyFilter - компонент для отображения условия фильтрации. Для работы компонента требуются следующие атрибуты:

  • property - свойство сущности или свойство вложенной сущности (например, customer.age).
  • operation - устанавливает операцию фильтрации: CONTAINS, EQUAL и т.д.
  • dataLoader - устанавливает идентификатор DataLoader.
