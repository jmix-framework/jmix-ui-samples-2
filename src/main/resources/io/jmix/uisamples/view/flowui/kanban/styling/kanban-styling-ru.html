Существуют различные способы стилизации компонента доски Kanban. Вы можете использовать свойства компонента, рендереры и классы CSS для стилизации.

Обратите внимание, что в данном примере для столбцов определён атрибут color. Он окрашивает заголовки.

Также, если определить атрибут applyColumnColorToTasks, то цвет столбца будет применяться к стилю карточки задачи. Если у задачи определен свой цвет, то цвет задачи будет иметь приоритет над цветом столбца.
