VectorSource может содержать любой тип геометрического объекта. В данном примере, полигоны добавляются в источник программным способом. Чтобы создать и отобразить полигон в компоненте GeoMap нужно использовать PolygonFeature.

Если полигоны хранятся в JPA или DTO сущности, то нужно использовать DataVectorSource. Он связывается с контейнером данных через dataContainer и property атрибуты. Свойство обязано иметь тип Polygon.
