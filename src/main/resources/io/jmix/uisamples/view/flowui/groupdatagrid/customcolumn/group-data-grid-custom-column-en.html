GroupDataGrid can group data items by non-persistent properties. To enable grouping, describe the property in the GroupDataGridItems. It requires a GroupPropertyDescriptor object where you can define the property name and its value provider.

Note that the property name must match the column key that is not bound to an entity property.

By default, sorting will not work for this property. To sort the data correctly, you can specify properties from an entity. For example, if you use BaseGroupPropertyDescriptor (the base implementation of GroupPropertyDescriptor), you can set sort properties using the withSortProperties() method. These properties will be used for sorting.
