Маркеры и точки, отображаемые в компоненте GeoMap, могут иметь различные стили отображения.

MarkerFeature по умолчанию имеет синюю иконку. Имеется возможность добавить дополнительный стиль, например, с отображением текста или заменить иконку на другую.

PointFeature может быть стилизован под отображение:

  • иконки
  • текста
  • любой формы объекта

PointStyle - опциональный вспомогательный класс. Он предоставляет методы стилизации, применимые только для Point.
