The Jmix Platform doesn't limit the possibility of integration with any Vaadin components, including addons. This way you can connect any addon, and it will work out of the box.

This example demonstrates the integration of the Charts add-on. You can find other addons here: Vaadin Directory

To connect the addon you must complete the following steps:

  1. Add the vaadin-addons maven repository to the project's build.gradle: 
    repositories {
    // ...
    maven {
        url 'https://maven.vaadin.com/vaadin-addons'
    }
    // ...
}
  2. Add the dependency of the add-on to the build.gradle: 
    dependencies {
    // ...
    implementation 'com.storedobject.chart:so-charts:3.2.4'
    // ...
}

To be able to use the components from add-on in views XML declaratively, a few more steps are needed:

  1. Create an XSD for the components (see chart.xsd
  2. Implement the component loaders (see ChartLoader.java).
  3. Register the components and its loaders in the application (see ChartComponentRegistration.java).

Note: the sample provides partial support for components (not all properties and components are described in XML).
