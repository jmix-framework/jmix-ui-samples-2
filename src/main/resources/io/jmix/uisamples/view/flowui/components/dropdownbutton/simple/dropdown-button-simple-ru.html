DropdownButton — кнопка в выпадающим списком элементов.

Кнопка может содержать текст или иконку.

Компонент имеет несколько вариантов вложенных элементов:

  • textItem - содержит текст. ClickListener может быть добавлен в контроллере.
  • componentItem - содержит пользовательский компонент. ClickListener также может быть добавлен в контроллере.
  • actionItem - может быть связан с действием через атрибут ref.
  • separator - разделитель для элементов.
