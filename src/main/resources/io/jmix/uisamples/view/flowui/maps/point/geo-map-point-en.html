Users can place points on the map.

To display points from JPA or DTO entity is used DataVectorSource. It binds with data container via dataContainer and property attributes. Property must have Point datatype or any other supported by GeoMap component. By default, all points from data container are displayed with blue marker.

If it is needed to display points that not related wth JPA or DTO, the VectorSource can be used. This source provides methods to add/remove features of any type.
