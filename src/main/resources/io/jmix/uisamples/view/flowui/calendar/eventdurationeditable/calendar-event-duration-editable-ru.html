Свойство eventDurationEditable позволяет изменять длительность событий. Длительность событий можно менять по их правой границе.

Свойство eventResizableFromStart позволяет изменять длительность событий по их левой границе. Если свойство eventDurationEditable выключено, события изменять нельзя.

Событие EventResizableEvent срабатывает когда пользователь изменяет длительность календарного события. Здесь можно сохранить сущность или любую другую информацию.
