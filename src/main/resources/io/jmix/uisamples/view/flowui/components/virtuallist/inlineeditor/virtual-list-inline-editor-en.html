An example how to build inline editor on entity with VirtualList. Here are the steps you need to take to do this:

  • In the view to declare VirtualList;
  • For the created VirtualList generate Renderer;
  • In the Renderer start to generate card of Vertical Layout, each contains following elements:
    • Avatar - optional element, stands first in the Vertical Layout (card);
    • Info Layout - any composition of elements that can display information in the card about the details of the current item;
    • Button Layout - layout where the control buttons for working with the current element are located;
  • Create a DetailView which edits the selected entity or creates a new one;
  • Hang action on all buttons to control CRUD;
  • Your inline editor with VirtualList is done.
