An example of how to build an inline editor on an entity using VirtualList. Here are the steps you need to take:

  • In the view, declare VirtualList;
  • Generate a Renderer for the created VirtualList;
  • In the Renderer, begin by generating cards of Vertical Layout, each containing the following elements:
    • Avatar - an optional element, placed first in the Vertical Layout (card);
    • Info Layout - any combination of elements that can display information about the current item;
    • Button Layout - a layout containing control buttons for working with the current element;
  • Create a DetailView to edit the selected entity or create a new one;
  • Attach actions to all buttons to control CRUD operations;
  • Your inline editor with VirtualList is ready.
