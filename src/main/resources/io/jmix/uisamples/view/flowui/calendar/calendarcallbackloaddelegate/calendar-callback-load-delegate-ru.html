callbackDataProvider поддерживает установку делегата загрузки данных, в случае когда использование XML не подходит по разным причинам. Делегат загрузки обязан вернуть список CalendarEvent.

Обратите внимание, если делегат возвращает список EntityCalendarEvent, то нужно обязательно указать соотношение свойств сущности и CalendarEvent в callbackDataProvider.
