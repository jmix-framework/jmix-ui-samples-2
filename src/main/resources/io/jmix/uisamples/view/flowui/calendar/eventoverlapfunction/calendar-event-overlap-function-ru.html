Пример демонстрирует как добавить правила для перекрытия календарных событий. Функция перекрытия принимает два параметра:

  • stillEvent - перекрываемое событие.
  • movingEvent - событие, которое перекрывает другое.

Дополнительные свойства находятся в свойство extendedProps. Путь к свойству через точку будет отформатирован единое свойство, которое объединяет остальные с заглавными буквами.
