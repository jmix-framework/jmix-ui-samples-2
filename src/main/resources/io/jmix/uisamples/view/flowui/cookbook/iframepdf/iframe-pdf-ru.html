В данном примере демонстрируется подход для отображения PDF-файлов в виде статического ресурса с использованием компонента IFrame.
Статические ресурсы должны хранится в папке META-INF/resources. Чтобы отображения таких ресурсов стало доступно - необходимо настроить директиву политики frame-ancestors.
См. исходный код в IframeCustomSecurity.

В исходном коде указана директива, которая разрешает использовать ресурсы в том случае, если запрос идёт с demo.jmix.io.
Более подробная информация доступна в документации к директиве: CSP: frame-ancestors - HTTP | MDN 6.
