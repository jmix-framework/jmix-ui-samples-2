The GeoMap supports displaying static images as a map content. There is an image layer that can display single images. It contains the following sources:

  • imageStaticSource displays static images. The url attribute defines the URL of external or application resource (see Vaadin docs :: Loading Resources) The source also supports programmatically setting an image using StreamResource.
  • imageWmsSource displays single, untiled images from WMS services.

In this example, GeoMap displays image from application resources. Note, that imageStaticSource and mapView must define projection and extent.
