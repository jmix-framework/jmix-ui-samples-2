Компонент fileStorageUploadField загружает файлы в систему хранения файлов. Он позволяет пользователям загружать файл и автоматически связывает его с атрибутом сущности как объект FileRef. Эта функциональность позволяет пользователям управлять хранением файлов и доступом к ним в приложении, причем FileRef служит ссылкой на расположение файла в системе хранения.

В этой реализации компонента fileStorageUploadField предусмотрена "зона перетаскивания", позволяющая загружать файлы с помощью действия перетаскивания в указанную область.
