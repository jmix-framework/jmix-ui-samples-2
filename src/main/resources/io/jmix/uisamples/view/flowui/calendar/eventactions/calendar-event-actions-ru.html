Вы можете подписаться на определённый набор событий, которые срабатывают при определённых действиях пользователя над календарным событием. В дополнение к событиям в примере, существуют события:

  • EventMouseEnterEvent - срабатывает, когда пользователь наводит указатель мыши на событие календаря.
  • EventMouseLeaveEvent - срабатывает, когда пользователь уводит указатель мыши с календарного события.
