GroupDataGrid имеет следующие специфические события:

  • GroupDataGridGroupingChangedEvent — происходит при изменении группировки.
  • CollapseEvent — происходит при сворачивании группы.
  • ExpandEvent — происходит при разворачивании группы.
  • GroupDataGridGroupItemClickEvent — происходит при щелчке по строке группы.
  • GroupDataGridGroupItemDoubleClickEvent — происходит при двойном щелчке по строке группы.
