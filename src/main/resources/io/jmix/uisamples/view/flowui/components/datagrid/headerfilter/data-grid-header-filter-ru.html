Атрибут filterable для вложенного элемента columns в DataGrid позволяет настраивать фильтры в заголовках колонок.

Колонка с фильтром имеет специальную кнопку, по нажатию на которую открывается всплывающее окно с компонентом propertyFilter для этой колонки.

<columns filterable="true"> добавляет фильтры для всех колонок. По умолчанию все фильтры скрыты.

Также можно указать атрибут filterable непосредственно для колонки. Значение в колонке имеет максимальный приоритет.
