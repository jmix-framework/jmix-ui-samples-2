На карте могут быть размещены точки.

Для отображения точек из JPA или DTO сущностей используется DataVectorSource. Он связывается с контейнером данных через dataContainer и property атрибуты. Свойство сущности обязано иметь тип Point или любой другой, который поддерживается компонентом GeoMap. По умолчанию все точки, получаемые из контейнера данных отображаются синем маркером.

Если необходимо отобразить точки, которые не связаны с JPA или DTO сущностью, можно использовать VectorSource. Данный источник предоставляет методы для добавления/удаления геометрических объектов любого типа.
