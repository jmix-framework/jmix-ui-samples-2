The context menu in the DataGrid component is accessible by default by right-clicking. By default, the context menu duplicates actions associated with the component. To disable automatic generation of context menu elements, you can use the showInContextMenuEnabled attribute in the actions element.

You can define your own context menu elements using the contextMenu element in the XML markup.

To add an icon for a context menu element, you must use the icon attribute. To automatically assign icons of related actions to context menu elements, you can use the application property: jmix.ui.component.grid-context-menu-show-action-icons.

It is also possible to customize the display of a context menu element using dynamicContentHandler.
