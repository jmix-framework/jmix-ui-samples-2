Существует возможность настроить отображения столбца.

Следующие renderer можно определить декларативно в XML:

  • localDateRenderer
  • localDateTimeRenderer
  • numberRenderer

Например:

<column property="dueDate"> <localDateRenderer format="yyyy-MM-dd"/> </column>

Также можно предоставить renderer в контроллере экрана с помощью аннотации@Supply.

Метод, помеченный аннотацией @Supply, возвращает значение, которое будет передано в качестве входного параметра методу, определенному subject.

Кроме того, вы можете использовать для этого метод addColumn().
