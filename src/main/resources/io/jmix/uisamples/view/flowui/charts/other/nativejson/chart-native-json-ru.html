nativeJson - это универсальное свойство, которое может использоваться для настройки компонента chart. Дополнительные настройки JSON могут быть переданы в качестве строки.

Рекомендуется использовать это свойство в том случае, если для необходимых настроек нет Java API или XML-разметки.

Примечание: nativeJson будет объединён с уже установленными свойствами.
Также, свойство nativeJson может перезаписать свойства, уже загруженные из XML, либо JAVA API, в том случае, если они пересекаются.

В данном примере основные свойства chart установлены в тэге nativeJson. Стоит отметить, что свойство title намеренно настраивается в нескольких местах, чтобы продемонстрировать, как компонент chart объединяет свои настройки и JSON.

При нажатии на подзаголовок chart можно перейти на исходный пример Apache ECharts.
