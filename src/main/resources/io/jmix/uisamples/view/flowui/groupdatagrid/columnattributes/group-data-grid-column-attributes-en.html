• groupIcon - specifies an icon to be displayed in the header of the group column. If you set the value to null, the default icon will be displayed.
  • groupIconVisible - specifies whether to display the icon in the group column header.
  • displayItemsCount - specifies whether to display the count of items in brackets for group rows.
  • displayColumnsGrouperOnIconClick - specifies whether to display the Columns Grouper when the icon is clicked.
