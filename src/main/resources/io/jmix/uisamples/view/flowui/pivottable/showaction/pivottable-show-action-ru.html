ShowPivotTableAction - действие, которое позоволяет показать данные из таблицы в отдельном экране с компонентом PivotTable. Данное действие можно использовать как из XML-дескриатора экрана, так и из его контроллера.

PivotTableViewBuilder - это Spring компонент. Он предоставляет API для настройки PivotTable в открывающемся экране.
