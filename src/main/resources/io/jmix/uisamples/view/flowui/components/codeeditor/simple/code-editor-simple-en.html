CodeEditor is a component to display and enter source code.

It is multi-line text area featured with code highlighting and optional print margin and gutter with line numbers.

UseSoftTabs property sets the tab character representing mode. Soft tabs mode means that tab character will be represented as a series of spaces. Hard tabs mode means that tab character will be represented as the special \t character.
TextWrap property wraps text to the next line automatically. This way, horizontal scroll bars will not appear in the input field.
