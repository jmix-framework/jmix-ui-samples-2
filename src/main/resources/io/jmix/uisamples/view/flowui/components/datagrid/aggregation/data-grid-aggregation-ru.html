Атрибут aggregatable включает агрегацию для строк DataGrid. Поддерживаются следующие операторы:

  • SUM - Вычисляет сумму
  • AVG - Находит среднее значение
  • COUNT - Вычисляет количество записей
  • MIN - Находит минимальное значение
  • MAX - Находит максимальное значение

Элемент aggregation должен быть установлен для агрегированных столбцов DataGrid с атрибутом type, который задает агрегирующую функцию. Агрегированные значения таблицы отображаются в дополнительной строке снизу DataGrid.

Атрибут aggregationPosition позволяет указать расположение строки агрегации:TOP или BOTTOM. BOTTOM используется по умолчанию.

В дополнение к операциям, перечисленным выше, вы можете определить собственную стратегию агрегации, используя AggregationStrategy.
