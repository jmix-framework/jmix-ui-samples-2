В данном примере, ломанные линии добавляются в источник программно. Чтобы создать ломаную линию и отобразить её в компоненте GeoMap, нужно использовать LineStringFeature.

Если ломанные линии хранятся в JPA или DTO сущности, то нужно использовать DataVectorSource. Он связывается с контейнером данных через dataContainer и property атрибуты. Свойство обязано иметь тип LineString.
