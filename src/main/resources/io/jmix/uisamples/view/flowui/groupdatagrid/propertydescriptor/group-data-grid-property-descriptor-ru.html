GroupDataGrid может группировать элементы данных по неперсистентным свойствам. Чтобы настроить группировку, опишите свойство в GroupDataGridItems. Для этого используется объект GroupPropertyDescriptor, в котором можно задать название свойства и поставщика его значения.

Обратите внимание, что название свойства должно совпадать с ключом столбца, который не связан со свойством сущности.

По умолчанию, сортировка по такому свойству не работает. Чтобы правильно сортировать данные, можно указать свойства из сущности. Например, если используется BaseGroupPropertyDescriptor (базовая реализация GroupPropertyDescriptor), то можно задать свойства сортировки с помощью метода withSortProperties(). Эти свойства будут использованы для сортировки.
