Для создания MultiSelectComboBox, связанного с атрибутом сущности, необходимо использовать атрибуты dataContainer и property.

dataContainer содержит экземпляр сущности, property указывает атрибут сущности для редактирования.

Также, компонент может брать список элементов из контейнера данных. Для этого используется атрибут itemsContainer.
