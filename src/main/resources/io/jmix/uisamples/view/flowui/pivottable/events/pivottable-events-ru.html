PivotTable два специфичных типа событий.

Событие PivotTableRefreshEvent вызывается, когда параметры компонента были изменены через элементы пользовательского интерфейса.

Событие PivotTableCellClickEvent вызывается при нажатии на ячейку таблицы. Данное событие возвращает все сущности, атрибуты которых использовались в аггрегации для данной ячейки.
