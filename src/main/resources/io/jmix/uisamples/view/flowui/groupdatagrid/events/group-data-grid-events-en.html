GroupDataGrid has the following specific events:

  • GroupDataGridGroupingChangedEvent - fired when grouping is changed.
  • CollapseEvent - fired when a group is collapsed.
  • ExpandEvent - fired when a group is expanded.
  • GroupDataGridGroupItemClickEvent - fired when a group row is clicked.
  • GroupDataGridGroupItemDoubleClickEvent - fired when a group row is double-clicked.
