Компонент доски Kanban имеет определенный набор событий, которые доступны для прослушивания. Чтобы подписаться на них: необходимо определить слушатель в контролере экрана декларативно, либо добавить его с помощью соответствующего метода компонента.

Некоторые события могут содержать в себе дополнительную информацию о контексте. Так, например, событие TaskClick несёт в себе информацию о сущности задачи, с которой связана карточка, на которую произошёл щелчек.
