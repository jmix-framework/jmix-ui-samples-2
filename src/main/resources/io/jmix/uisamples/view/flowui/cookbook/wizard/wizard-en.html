Wizard is a component for step-by-step data filling.

This example demonstrates the wizard for creating the Employee entity (see Employee.java).
The wizard opens in the dialog by clicking the button and consists of three steps:

  • Filling in basic information about the employee
  • Filling in the address
  • Checking the entered information

Each step is associated with data, information about which is stored in the main view WizardDialog. The first two steps have fields that will be validated when moving to the next step.

The user can switch between steps at any time (if the data at the current step is correct) using the corresponding buttons, which change their behavior and state depending on the current step.

The wizard is based on the tabSheet component. The steps are implemented as separate tabs. The content of each tab is implemented as a fragment.
