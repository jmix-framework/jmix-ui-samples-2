Свойство eventConstraintGroupId устанавливает название группы, которое должно быть использовано для ограничения календарных событий в перемещении и изменении их длительности. Данное свойство применяется ко всем событиям независимо от поставщиков данных.

Другой способ указать ограничение — использовать свойство eventConstraintBusinessHoursEnabled. Если это свойство включено, компонент будет придерживаться указанных рабочих часов в качестве ограничения для календарного события.

Календарные события, имеющие одинаковый идентификатор группы, будут перемещаться вместе.
