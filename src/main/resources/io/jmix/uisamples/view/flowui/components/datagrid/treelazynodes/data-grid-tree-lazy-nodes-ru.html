Этот пример демонстрирует один из вариантов реализации "ленивой" загрузки данных для компонента TreeDataGrid.

TaskLazyTreeDataGridItems - источник данных, который определен в контроллере экрана. Этот источник данных загружает их из базы данных, используя компонент DataManager.

Когда узел иерархии раскрывается, необходимые данные догружаются из базы.
