The fileStorageUploadField component uploads files to the Jmix FileStorage. It allows users to upload a file and automatically links it to an entity attribute as a FileRef object. The FileRef serves as a reference to the file's location in the file storage.

This example allows uploading of files of any type with a size limit of 50 KB. The component features a "drop zone" for drag-and-drop uploads, activated by dragging files over the component's button.

Note: In this example, files are uploaded to the temporary storage and immediately removed. See the comments in FileStorageUploadFieldSample.java.
