Список элементов ComboBox может быть задан произвольно с помощью следующих методов:

  • setItems() - позволяет программно задать список элементов компонента;

  • ComponentUtils.setItemsMap() - позволяет явно задать строковые описания значениям элементов;

Атрибут placeholder позволяют задать строку, отображаемую вместо значения.
