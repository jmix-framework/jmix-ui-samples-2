Каскадные выпадающие списки фильтруют свои элементы на основе предыдущего выбора в иерархии.

Данный пример показывает реализацию выбора двух зависимых сущностей: планет и их спутников. Когда пользователь выбирает планету в первом списке, второй список отображает спутники этой планеты.

Сущности загружаются в два контейнера данных: planetsDc и moonsDc. Запрос загрузчика контейнера спутников содержит параметр :container_planetsDc, который ссылается по имени на контейнер планет. Этот параметр распознается фасетом DataLoadCoordinator, который автоматически перезагружает контейнер спутников каждый раз, когда изменяется выбранное значение в контейнере планет.

Обработчик onPlanetsComboBoxValueChange() очищает текущее значение списка спутников когда изменяется выбранная планета.
