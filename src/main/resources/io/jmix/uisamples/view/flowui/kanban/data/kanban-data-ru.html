Компонент доски Kanban имеет возможность привязки данных. Для этого следует указать атрибут dataContainer. Вы можете фильтровать и редактировать данные в контейнере, все изменения будут отображены в компоненте.

Чтобы задать соответствие атрибутов отображаемой сущности с атрибутами карточки задачи Kanban необходимо определить объект propertiesMapping.

Карточка задачи Kanban имеет следующий набор атрибутов:

  • id (обязательный)
  • status (обязательный)
  • text (обязательный)
  • username
  • userAvatar
  • color
  • dueDate
  • progress
  • tags
  • priority
  • swimlane

Правилами отображения соответствующих элементов на карточки задачи Kanban можно управлять с помощью свойств компонента. Обязательные атрибуты имеют стандартные значения. Если объект propertiesMapping не указан, при определении соответствия в сущности будут использоваться поля id, status и text.
