aggregatable attribute enables aggregation for DataGrid rows. The following operations are supported:

  • SUM - calculate the sum
  • AVG - find the average value
  • COUNT - calculate the total number
  • MIN - find the minimum value
  • MAX - find the maximum value

The aggregation element should be set for aggregated DataGrid cells with the type attribute, which sets the aggregation function. The aggregated table values are shown in an additional row at the bottom of the DataGrid.

The aggregationPosition attribute allows you to specify the location of the aggregation row: TOP or BOTTOM. BOTTOM is used by default.

In addition to the operations listed above, you can define a custom aggregation strategy by implementing the AggregationStrategy interface.
