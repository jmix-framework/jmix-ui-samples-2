Компонент MultiValuePicker работает со списком значений любого типа. Он состоит из текстового поля и набора кнопок, определяемых действиями.

MultiValuePicker удобно использовать, если вам нужно редактировать список значений с помощью действий, а не в текстовом поле.

Вы можете определить пользовательские и предопределенные действия для MultiValuePicker, отображаемые в виде кнопок справа.

Framework предоставляет предопределенные действия value_clear и multi_value_select.

Действие multi_value_select устанавливает значение для MultiValuePicker с помощью диалогового окна выбора значения. Диалоговое окно выбора значения генерирует определенный компонент для выбора или создания значения в зависимости от типа переданного значения.
