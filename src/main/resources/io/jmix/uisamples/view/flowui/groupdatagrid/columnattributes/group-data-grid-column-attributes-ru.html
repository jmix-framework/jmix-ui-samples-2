• groupIcon - задаёт иконку, которая должна отображаться в заголовке группового столбца. Если задать значение null, будет показана иконка по умолчанию.
  • groupIconVisible - указывает, отображать ли иконку в заголовке группового столбца.
  • displayItemsCount - указывает, отображать ли количество элементов в скобках для групповых строк.
  • displayColumnsGrouperOnIconClick - указывает, показывать ли группировщик столбцов по клику на иконку.
