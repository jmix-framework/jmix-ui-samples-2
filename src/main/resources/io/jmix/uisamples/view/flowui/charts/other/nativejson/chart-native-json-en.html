nativeJson is a universal property that can be used to configure the chart component. An additional JSON options can be set as a string.

Most often, this property is suitable for cases where there is on Java API or XML markup for the required options.

Note: nativeJson will be merged with existing properties.
Also, this property can override options loaded from XML or from component Java API if they overlap.

In this sample, the main chart options are set in XML nativeJson tag. It's worth noting that the title property is configured from different places to demonstrate that chart is merging options from JSON.

When you click on the chart subtitle you will see the original Apache ECharts example.
