Компонент EntityPicker предназначен для отображения экземпляра сущности и выполнения действий, представляемых кнопками в правой части поля.

Компонент может быть связан с атрибутом сущности путем указания XML атрибутов dataContainer и property. Атрибут сущности должен быть ссылкой на другую сущность.
