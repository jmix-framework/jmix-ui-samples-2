You can use any third-party Vaadin add-ons located in the Vaadin Directory or elsewhere. This example demonstrates the integration of the SO Charts add-on.

To include the Vaadin add-on in your application, add the Vaadin Add-ons maven repository and the add-on dependency to the project's build.gradle:

repositories {
    maven {
        url 'https://maven.vaadin.com/vaadin-addons'
    }
    // ...
}

dependencies {
    implementation 'com.storedobject.chart:so-charts:3.2.4'
    // ...
}
After that, you can use the add-on components programmatically, see for example ChartsAddonSample.java.

To be able to use the add-on components in views XML declaratively, a few more steps are needed:

  1. Create an XSD for the components (see chart.xsd)
  2. Implement the component loaders (see ChartLoader.java).
  3. Register the components and its loaders in the application (see ChartComponentRegistration.java).

Note: This example provides only partial support for the add-on components (not all components and properties are described for usage in XML).
