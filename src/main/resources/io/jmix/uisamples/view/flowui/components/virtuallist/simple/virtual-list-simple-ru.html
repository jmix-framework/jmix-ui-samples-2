VirtualList позволяет отображать длинный список элементов внутри прокручиваемого контейнера без ущерба для производительность.

Каждый элемент отображается в процессе прокручивания списка.

Пользовательский renderer для элементов списка можно установить в контроллере экрана, используя Java код. Также, можно указать средство визуализации с помощью ValueProvider, если мы хотим отображать простой текст.

Для всего этого вы можете использовать метод setRenderer() или аннотацию @Supply.
