В данном примере, все объекты добавляются в источник программно. Чтобы создать multipoint, multipolygon или multi linestring и отобразить его в компоненте GeoMap, нужно использовать следующие типы объектов:

  • MultiPointFeature
  • MultiPolygonFeature
  • MultiLineStringFeature

Если подобные объекты хранятся в JPA или DTO сущности, то следует использовать DataVectorSource. Он связывается с контейнером данных через dataContainer и property атрибуты. Свойство сущности должно иметь один из следующих типов:

  • MultiPoint
  • MultiPolygon
  • MultiLineString
