У PivotTable есть специальный режим, в котором отображаются элементы пользовательского интерфейса для манипулирования данными. За эту функциональность отвечает атрибут showUI. Если сделать PivotTable неактивным с помощью вызова PivotTable#setEnabled(false), то его элементы управления также станут неактивными.

Следующие атрибуты могут быть применены к любому режиму, но только для табличных рендеров:

  • showRowTotals отображает сумму по строке;
  • showColTotals отображает сумму по столбцу.
