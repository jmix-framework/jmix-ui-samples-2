SimplePagination can manage the maximum size of the page. This functionality is presented by the following properties:

  • itemsPerPageVisible - defines whether the dropdown list of maximum page size items should be visible.
  • itemsPerPageItems - sets custom maximum page size items that should be used in the dropdown list. Values should be separated by the comma, e.g. "10, 25, 50".
  • itemsPerPageDefaultValue - sets the default value from available items that should be used by default. It can be used even if the itemsPerPageVisible attribute is set to "false".
  • itemsPerPageUnlimitedItemVisible - defines should be item that is shown as empty item in Select visible. If this item is selected the maximum page size will be provided by the entityPageSize application property.
