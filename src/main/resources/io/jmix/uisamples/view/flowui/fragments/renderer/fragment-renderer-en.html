The virtualList component supports fragments as renderers.

To do this you need to add a nested <fragmentRenderer/> element. Using the class attribute, you must specify the fragment class that extends the special io.jmix.flowui.fragmentrenderer.FragmentRenderer abstract class.

Fragment renderer is very similar to a regular fragment. The difference is that the fragmentRenderer must contain an instance container in which the item to be renderer will be set. To specify which container an item should be placed in, use the @RendererItemContainer annotation.

List of supported components:

  • VirtualList
  • DataGrid.Column
  • CheckboxGroup
  • MultiSelectComboBox
  • MultiSelectComboBoxPicker
  • EntityComboBox
  • RadioButtonGroup
  • Select
  • ListBox
  • MultiSelectListBox
