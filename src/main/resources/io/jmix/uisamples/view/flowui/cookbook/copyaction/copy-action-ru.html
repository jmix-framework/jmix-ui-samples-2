В данном примере показаны возможные пути реализации копирования сущностей в экране просмотра.

В экране определены два действия: copy и copyAndEdit.

Действие copy создает новый экземпляр сущности и сразу же сохраняет его в базу данных.

Действие copyAndEdit создает новый экземпляр сущности и открывает экран редактирования, который сохраняет новый экземпляр только когда пользователь нажимает OK.

Оба действия используют метод MetadataTools.copy() для "поверхностного" копирования выбранного экземпляра.
