LineStringFeature позволяет стилизовать только линию: её цвет, ширину, пунктир, и т.д.

Первый LineString не имеет стилей. Второй же добавляет два стиля и при этом последний перекрывает собой предыдущий.

LineStringStyle - опциональный вспомогательный класс. Он предоставляет методы стилизации, применимые только для LineString.
