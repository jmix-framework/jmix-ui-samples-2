Доска Kanban предназначена для визуального отображения работы на различных этапах процесса с использованием карточек для представления задач, и столбцов для представления каждого этапа процесса. Компонент основан на веб-компоненте Smart HTML Elements - Kanban.

Так как компонент используется для визуализации набора сущностей, его функциональность несколько схожа с компонентом DataGrid. Так, например, в компонент могут быть интегрированы действия создания, редактирования и удаления сущностей.

Помимо этого в компоненте реализована привязка данных. Вы можете фильтровать данные с использованием GenericFilter компонента. Все изменения набора данных будут отображены на доске Kanban.
