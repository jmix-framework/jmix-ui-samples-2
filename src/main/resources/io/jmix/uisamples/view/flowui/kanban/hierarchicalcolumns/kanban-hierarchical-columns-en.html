The Kanban board component supports hierarchical columns. To define a hierarchical column, simply add the column to an existing column.

There are two column display modes:

  • TABS - display columns by tabs
  • COLUMNS - display all columns within a parent column

When using the TABS mode, you can control the default selected column using the selected attribute.

Note: hierarchical columns are not displayed if the Kanban board has swimlanes defined.
