To configure multiple sorting of columns in DataGrid three attributes are used:

  • multiSort - enables or disables multiple sorting
  • multiSortOnShiftClickOnly - allows user to add the selected column to the sort only when the Shift key is pressed
  • multiSortPriority - defines the rule by which subsequent columns will be added to the sort. PREPEND - to the beginning, APPEND - to the end. PREPEND by default.
