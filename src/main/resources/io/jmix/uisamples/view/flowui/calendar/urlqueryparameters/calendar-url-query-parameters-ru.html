Аддон Calendar предоствляет параметры для фасета UrlQueryParametersFacet. Он содержит следующие параметры:

  • calendarDisplayMode - устанавливет режим отображения календаря. Может быть изменено на любое другое имя.
  • calendarDate - навигирует на указанную дату. Может быть изменено на любое другое имя.
