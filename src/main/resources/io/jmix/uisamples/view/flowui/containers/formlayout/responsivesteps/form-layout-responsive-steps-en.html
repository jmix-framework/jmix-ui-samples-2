responsiveSteps are used to determine the visual style of the formLayout depending on the size of the container.

The following attributes can be configured:

  • minWidth - the minimum width for which the selected style will be applied.
  • columns - the number of columns that will be displayed in the selected step.
  • labelsPosition - an optional parameter that specifies how labels are displayed.

Please note: to use ASIDE labels, you need to wrap the component in a formItem.

formItem - a special wrapper for formLayout, which controls the position of the labels and the display of the component in the form.

To configure the label, enabled and visible attributes, use formItem
