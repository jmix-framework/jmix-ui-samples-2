In this example, all features added to the source programmatically. To create multipoint, multipolygon or multi linestring and display it in GeoMap component the following features can be used:

  • MultiPointFeature
  • MultiPolygonFeature
  • MultiLineStringFeature

If feature stored in JPA or DTO entity, the DataVectorSource should be used. It binds with data container via dataContainer and property attributes. Property must have on of the following types:

  • MultiPoint
  • MultiPolygon
  • MultiLineString
