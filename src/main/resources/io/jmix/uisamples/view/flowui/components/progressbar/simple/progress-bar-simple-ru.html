Компонент ProgressBar служит для отображения хода выполнения некоторого длительного процесса.

В данном примере ProgressBar обновляется фоновой задачей.

C помощью атрибута indeterminate можно задать отображение неопределенного состояния индикатора. Если значение атрибута равно true, то индикатор отображает неопределенное состояние. По умолчанию false.
