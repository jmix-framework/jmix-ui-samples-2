Series that are in the same stack will be drawn on top of each other.

The stack property determines the name of the stack where the current series will set.

The stackStrategy property determines how to stack values when the stack property is set.

Available options:

  • SAME_SIGN - only stack values if the value to be stacked has the same sign as the currently cumulated stacked value.
  • ALL - stack all values, irrespective of the signs of the current or cumulative stacked value.
  • POSITIVE - only stack positive values.
  • NEGATIVE - only stack negative values.
