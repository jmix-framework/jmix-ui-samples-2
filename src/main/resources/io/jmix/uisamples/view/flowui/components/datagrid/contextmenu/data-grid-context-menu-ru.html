Контекстное меню в компоненте DataGrid доступно по умолчанию на правый щелчок мыши. По стандарту, в контекстное меню дублируются действия, связанные с компонентом. Чтобы выключить автоматическую генерацию элементов контекстного меню можно воспользоваться атрибутом showInContextMenuEnabled в элементе actions.

Вы можете определять собственные элементы контекстного меню с помощью элемента contextMenu в XML разметке.

Чтобы добавить иконку для элемента контекстного меню необходимо воспользоваться атрибутом icon. Чтобы элементам контекстного меню автоматически проставлялись иконки связанных действий можно воспользоваться свойством приложения: jmix.ui.component.grid-context-menu-show-action-icons.

Также, возможно определять отображение элемента контекстного меню с помощью dynamicContentHandler.
