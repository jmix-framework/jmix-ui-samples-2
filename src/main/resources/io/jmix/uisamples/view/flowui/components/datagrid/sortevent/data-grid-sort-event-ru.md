`com.vaadin.flow.data.event.SortEvent` реагирует на изменения в сортировке колонок.
Пример выше показывает текущий порядок сортировки для каждой колонки по мере его изменения. Чтобы добавить
собственное поведение при изменении сортировки — например, инициировать другие обновления на экране — добавьте свою собственную логику,
используя `event.getSortOrder()` для доступа к порядку сортировки.
