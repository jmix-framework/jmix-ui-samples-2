Компонент fileStorageUploadField загружает файлы в Jmix FileStorage. Он позволяет пользователям загружать файл и автоматически связывает его с атрибутом сущности в виде объекта FileRef. При этом FileRef служит ссылкой на расположение файла в FileStorage.

Данный пример позволяет загружать файлы любого типа размером до 50 КБ. Компонент содержит "зону перетаскивания" для загрузки файлов с помощью перетаскивания на кнопку компонента.

Примечание: В данном примере файлы загружаются во временное хранилище и сразу же удаляются. См. комментарии в файле FileStorageUploadFieldSample.java.
