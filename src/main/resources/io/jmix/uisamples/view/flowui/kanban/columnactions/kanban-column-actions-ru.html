Компонент доски Kanban поддерживает динамическое изменение настроек столбцов. Для управления этой функцией используйте атрибуты collapsible, columnReorderingAllowed и columnActionsEnabled.

Атрибут collapsible добавляет кнопки сворачивания и разворачивания столбцов в заголовок. Сворачивание также доступно по двочному щелчку по заголовку.

Атрибут columnReorderingAllowed разрешает изменять порядок столбцов с помощью Drag'n'Drop.

Атрибут columnActionsEnabled добавляет кнопку, которая открывает диалоговое меню, которое позволяет скрывать, раскрывать и сворачивать столбцы. Соответствующие действия доступны в Java API компонента.

Все описанные выше настройки сохарняются в настройках экрана для текущего пользователя, если добавлен SettingsFacet. При переоткрытии экрана настройки применятся вновь. В настройках экрана пользователя сохраняется порядок столбцов и их состояние (видимость и состояние свёрнутости).
