There are different ways to style the Kanban board component. You can use component properties, renderers, and CSS classes to style it.

Note that in this example, the color attribute is defined for the columns. Colors will be applied to the headers.

Also, if you define the applyColumnColorToTasks attribute, the column color will be applied to the task card style. If the task has its own color, the task color will take precedence over the column color.
