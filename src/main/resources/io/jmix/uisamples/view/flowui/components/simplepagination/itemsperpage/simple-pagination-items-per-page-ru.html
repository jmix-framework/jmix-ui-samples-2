SimplePagination может управлять максимальным количеством элементов на странице. Данная функциональность представлена следующими свойствами:

  • itemsPerPageVisible - определяет должен ли быть видимым выпадающий список с вариантами максимального количества элементов на странице.
  • itemsPerPageItems - устанавливает список максимальных значений страницы. Значения должны быть разделены запятой, например: "10, 25, 50".
  • itemsPerPageDefaultValue - устанавливает значение по умолчанию из списка доступных значений. Может использоваться если атрибут itemsPerPageVisible установлен в "false".
  • itemsPerPageUnlimitedItemVisible - определяет должно ли значение, которое отображается как пустая строка в компоненте Select быть видимым. Если данное значение выбрано, то максимальный размер страницы будет равен значению свойства приложения entityPageSize.
