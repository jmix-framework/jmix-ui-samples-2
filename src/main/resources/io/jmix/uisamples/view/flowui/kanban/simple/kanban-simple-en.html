The Kanban board is designed to visually display work at various stages of a process using cards to represent tasks, and columns to represent each stage of the process. The component is based on the Smart HTML Elements - Kanban web-component.

Since the component is used to visualize a set of entities, its functionality is somewhat similar to the DataGrid component. For example, the component can contain actions for creating, editing, and deleting entities.

In addition, the component implements data binding. You can filter data using the GenericFilter component. All changes to the data set will be displayed on the Kanban board.
