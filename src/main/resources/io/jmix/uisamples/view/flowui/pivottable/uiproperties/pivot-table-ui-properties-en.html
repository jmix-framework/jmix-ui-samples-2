PivotTable has a special mode that displays UI elements for data manipulation. The showUI attribute is responsible for this functionality. If PivotTable is disabled with method PivotTable#setEnabled(false) call, its UI controls are also disabled.

The following attributes can be applied on any mode, but only for the table renderers:

  • showRowTotals shows and hides row totals;
  • showColTotals shows and hides column totals.
