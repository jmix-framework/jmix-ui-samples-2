The fileUploadField component provides a way to upload files. It enables users to select and upload a file, automatically saving the file content as a byte array to a specified entity attribute. This feature is useful for handling file data within the application's data model.

This example allows uploading of image files (PNG, JPEG) with a maximum size of 50 KB. The name of the uploaded file appears next to the upload button and also functions as a download link for quick access. The component features a "drop zone" for drag-and-drop uploads by dragging files over the component's button.
