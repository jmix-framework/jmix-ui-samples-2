Компонент virtualList поддерживает фрагменты в качестве рендереров.

Для этого вам необходимо добавить вложенный элемент <fragmentRenderer/>. Используя атрибут class, вы должны указать класс фрагмента который расширяет специальный абстрактный класс io.jmix.flowui.fragmentrenderer.FragmentRenderer.

Рендерер фрагменты очень похож на обычный фрагмент. Разница в том, что fragmentRenderer должен содержать контейнер, в который будет установлен элемент для рендеринга. Чтобы указать, в какой контейнер следует поместить элемент, используйте аннотацию @RendererItemContainer.

Ниже представлен список поддерживаемых компонентов:

  • VirtualList
  • DataGrid.Column
  • CheckboxGroup
  • MultiSelectComboBox
  • MultiSelectComboBoxPicker
  • EntityComboBox
  • RadioButtonGroup
  • Select
  • ListBox
  • MultiSelectListBox
