Для компонента доски Kanban возможно определить столбцы используя перечисление. Чтобы это сделать необходимо указать атрибут columnsEnum для элемента <columns/>. Таким образом все значения перечисления будет соотнесены со столбцами, при этом заголовок столбца будет автоматически проставлен локализованным значением перечисления.

Иерархичность колонок в случае определения с помощью перечисления не поддерживается.
