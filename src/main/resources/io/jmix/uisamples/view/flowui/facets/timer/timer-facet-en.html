Timer is a facet (non-visual component) designed to run UI code at specified time intervals.

The timer action is initiated from the browser as a user request and its handler can update UI components. The timer stops working when the view it was created for is closed.

Timers are defined using the timer element. It has the following attributes:

  • delay - defines timer interval in milliseconds;
  • autostart - if set to true, the timer starts immediately after the view is opened. By default, the value is false, witch means that the timer will start only when its start() method is invoked;
  • repeating - turns on repeated execution of the timer. If the attribute is set to true, the timer runs in cycles at equals intervals defined in the delay attribute. Otherwise, the timer runs only once - delay milliseconds after the timer start.

To execute code on timer, subscribe to its TimerActionEvent in the view controller.
