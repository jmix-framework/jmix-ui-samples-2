Данный пример показывает, как создать DataGrid с переменным набором колонок.

Он основан на key-value сущностях и контейнерах:

  • Данные предоставляются в виде списка экземпляров KeyValueEntity.
  • KeyValueCollectionContainer создается динамически и конфигурируется вызовами метода addProperty() чтобы соответствовать свойствам key-value сущностей.
  • DataGrid создается динамически с колонками, соответствующими свойствам контейнера.

См. исходный код в DataGridVariableColumnsSample.java
