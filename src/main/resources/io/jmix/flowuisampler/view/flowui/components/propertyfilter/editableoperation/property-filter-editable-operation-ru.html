Атрибут operationEditable определяет должен ли отображаться компонент выбора операции. Если значение true, пользователь сможет выбрать операцию фильтрации во время работы приложения. Список доступных операций зависит от типа свойства.

PropertyFilter позволяет подписаться на событие OperationChangeEvent. Оно срабатывает когда пользователь меняет операцию фильтрации.
