Scroller - a container that can scroll its content if it does not fit the size of the container.

The components direction can be defined by scrollBarsDirection attribute:

  • VERTICAL
  • HORIZONTAL
  • BOTH
  • NONE

scrollBarsDirection attribute allows configuring scroll bars. It can be horizontal, vertical - for horizontal and vertical scrolling respectively, both - for scrolling in any direction. Setting the value to none forbids scrolling in any direction.
