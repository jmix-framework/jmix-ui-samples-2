DropdownButton is a button with drop-down list of items.

The DropdownButton can contain text or icon.

The component has several items variants:

  • textItem - contains text. ClickListener can be added from a controller.
  • componentItem - contains a custom component. ClickListener can also be added from a controller.
  • actionItem - can be associated with an action using the ref attribute.
  • separator - just for separating items.
