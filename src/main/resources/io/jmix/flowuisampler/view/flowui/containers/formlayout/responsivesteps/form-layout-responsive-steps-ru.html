responsiveSteps используются для определения вида отображения formLayout в зависимости от размеров контейнера.

Можно настроить следующие атрибуты:

  • minWidth - минимальная ширина, при которой будет применен текущий шаг.
  • columns - количество отображаемых колонок на текущем шаге.
  • labelsPosition - опциональный параметр. Настраивает правило отображения заголовков для компонентов.
