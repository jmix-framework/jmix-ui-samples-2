multiSelectComboBox позволяет пользователям выбирать один или несколько элементов из выпадающего списка.

Атрибут clearButtonVisible позволяет пользователям отчистить все выбранные элементы.

multiSelectComboBox поддерживает следующие функции стандартного comboBox:

  • Custom items
  • Custom filtering
  • Handling user input
  • Custom renderer
