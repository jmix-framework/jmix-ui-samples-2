Чтобы начать пользоваться диалогом сообщений, просто напишите: dialogs.createMessageDialog().

Самый простой вариант использования диалогового окна сообщений — это отображение некоторого сообщения пользователю.

HTML форматирование текста, также, доступно: диалоговое окно может отображать содержимое HTML с помощью метода withContent().

Также доступны различные настройки диалога.
