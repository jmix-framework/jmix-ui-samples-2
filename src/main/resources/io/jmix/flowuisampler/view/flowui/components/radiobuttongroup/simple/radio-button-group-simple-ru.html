RadioButtonGroup обеспечивает выбор одного значения из списка опций, используя группу переключателей (радио-кнопок).

Атрибут themeNames задает расположение элементов группы. По умолчанию элементы располагаются по горизонтали.

В данном примере компонент используется для выбора значения перечисления (enum) для атрибута сущности.
