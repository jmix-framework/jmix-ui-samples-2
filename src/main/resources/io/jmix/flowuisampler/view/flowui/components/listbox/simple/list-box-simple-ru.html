ListBox позволяет пользователю выбирать значения из прокручиваемого списка элементов.

ListBox лучше всего подходит для использования в качестве облегченного, прокручиваемого списка из одного столбца для одиночного или множественного выбора элементов.

Вы можете использовать разделители для группировки связанных элементов.
