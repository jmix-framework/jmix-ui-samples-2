DataGrid is designed to display and sort tabular data, and provides means to manipulate rows.

In this example, you can play with the following DataGrid settings:

  • sortable - enables or disables the column sorting. The default value is true. When the sorting is enabled, the click on the column name will display the sorting icon to the right of the column header. Sorting of any specific column can be disabled by this column's sortable attribute.
  • resizable - enables or disables the column resizing. The default value is false.
  • columnReorderingAllowed - defines whether a user can change the columns order by dragging them with a mouse. The default value is false.
