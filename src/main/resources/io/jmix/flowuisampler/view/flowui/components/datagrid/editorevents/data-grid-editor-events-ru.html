DataGrid имеет несколько событий для прослушивания:

  • EditorOpenEvent - публикуется когда редактор DataGrid открывается.
  • EditorCancelEvent - публикуется когда редактирование отменено.
  • EditorCloseEvent - публикуется когда редактор DataGrid закрывается.
  • EditorSaveEvent - публикуется когда редактор DataGrid сохраняет изменения.
