tooltip - это настраиваемый вариант контекстной подсказки.

Атрибуты:

  • focusDelay - задержка в миллисекундах перед открытием всплывающей подсказки при фокусе клавиатуры, если не в ручном режиме.
  • hideDelay - задержка в миллисекундах перед закрытием всплывающей подсказки при потере наведения, если не в ручном режиме. При потери фокуса всплывающая подсказка сразу закрывается.
  • hoverDelay - задержка в миллисекундах перед открытием всплывающей подсказки при наведении, если не в ручном режиме.
  • manual - в значении true, всплывающая подсказка управляется программно, а не реагирует на фокус и события мыши.
  • position - положение всплывающей подсказки относительно компонента.
  • text - строка, используемая в качестве содержимого всплывающей подсказки.
