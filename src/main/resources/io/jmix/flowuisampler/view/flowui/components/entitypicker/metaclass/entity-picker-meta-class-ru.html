Компонент EntityPicker можно использовать без непосредственной привязки к данным, то есть без указания атрибутов dataContainer и property. В этом случае для указания типа сущности, с которой должен работать EntityPicker, используется атрибут metaClass.

В данном примере компонент работает с сущностью Customer, имя класса метаданных которой - sampler_Customer.
