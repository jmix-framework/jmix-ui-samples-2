В данном примере конфигурация была создана во время разработки и установлена "по умолчанию" с помощью атрибута default. Поэтому конфигурация появляется сразу после открытия экрана.

Условия конфигурации образуют древовидную структуру. Все условия верхнего уровня объединяются по логическому условию (по умолчанию И). Вы можете добавить группы И или ИЛИ для создания сложных условий.

Пользователи могут создавать групповые условия, выбрав пункт Создать Групповое условие диалога Добавить условие поиска. Групповые условия также могут быть определены в XML экрана (groupFilter).

Расширенный редактор условий доступен через пункт Редактировать выпадающего меню настроек фильтра.
